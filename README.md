# Testando Websocket


Passos:

- Clone esse projeto
- Instale as dependências: 

```sh

npm install

# Ou

yarn 

# ou

pnpm install

```

- Execute o projeto

```sh

npm start

```

O servidor da api iniciará em: 

```sh

# Api Servidor http
https://localhost:4000

# Websocket
http://localhost:8080

```

Para checar a api

```sh

curl --location 'http://localhost:4000/health-check'

```
